## 策略2 - 聚宽多策略组合（ETF + 小市值 + 白马）

- 代码位置：`@celue/basesuccess/etf白马小盘/index.pyt`，运行环境为 **聚宽（JoinQuant）**。
- 资金分配：通过 `STRATEGY_ALLOCATION` 将总资金按比例拆分为三部分：
  - ETF 轮动：40%（`'etf': 0.4`）
  - 小市值 399100 策略：35%（`'small_cap': 0.35`）
  - 大小择时白马策略：25%（`'white_horse': 0.25`）
- 统一调度：在 `initialize` 中调用 `schedule_tasks(context)`，为三类子策略分别注册 `run_daily` / `run_weekly` / `run_monthly` 任务。
- 持仓归属：使用 `g.stock_strategy` 与 `g.strategy_holdings` 记录每只股票属于哪条子策略，卖出时回写到对应的策略净值。
- ETF 轮动：
  - ETF 池由 `ETF_CONFIG['etf_pool']` 定义，支持固定周期动量与动态周期动量（基于 ATR 自适应 lookback）。
  - 每次从动量评分最高的 ETF 中选出 `target_num` 只（默认 1 只）作为持仓目标。
  - 仓位按该策略资金份额等权分配，超出目标的 ETF 会被卖出。
- 小市值策略（399100）：
  - 基于中证 1000（`SMALL_CAP_CONFIG['market_index']`）成分股，按流通市值与总市值筛选小盘股，并做行业分散。
  - 过滤次新股、ST、停牌、涨跌停，保留价格不高于上限的标的。
  - 每周二调仓：卖出不在新目标池且非昨日涨停的持仓，按小市值策略资金份额等权买入新标的。
  - 止损机制：既有大盘暴跌止损（指数跌幅超阈值时全清），也有个股跌破成本一定比例的个股止损。
  - 特定“空仓月份”可自动切换为持有指定 ETF（如 `'600036.XSHG'` 所在的列表）。
- 白马策略：
  - 通过大盘/小盘表现（沪深 300 与创业板相关指数）判断当前市场温度与风格偏好（大盘、小盘或外盘 ETF）。
  - 根据信号在大盘股池或小盘股池中按基本面与估值因子（PB、ROA、现金流、盈利增速等）选出白马股。
  - 按白马策略资金份额等权买入，支持多次止损与“补跌买入”逻辑（在止损后对表现最差的部分持仓做小额加仓以博反弹）。
  - 对昨日涨停股有单独的“涨停打开”监控逻辑，盘中打开涨停会择机卖出。

> 提示：本 README 仅做高层说明，具体细节（如调仓规则、选股 SQL/Query、止损参数）请直接阅读对应的 `index.pyt` 源码并结合聚宽回测界面验证。
